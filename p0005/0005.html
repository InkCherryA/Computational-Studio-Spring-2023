VR Body
Player Full Body Test

Following a tutorial on Youtube, we set up an rigged body with 3D model downloaded from Unity Asset Store.
The first step will be to import animation rigging from unity packages.
There are many components in the package that can help us set up the rigging for us, by simply clicking on "Rig Setup" and "Bone Renderer Setup".
Here you go, we can see the bones of the rigging, and all parts are connected to each other. Even if we pull on some of his body parts, it will still be connected.
However, the arms are not connected as we expected. Following the tutorial, we added "Two Bones IK Constraint" to help us determine the position of the arms while we are moving the hand.
Before we go into our VR body, we need to set Constraints for the VR controllers, aligning them with the body. For this part, we need to write our own script.
Now we can move around in our VR body without any problem.
