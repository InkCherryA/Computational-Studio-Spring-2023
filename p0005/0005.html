VR Body
For more Realistic feeling of VR, we need to make a player body that all parts are connected to each other instead of just a head and void hands. This project is to make a body that can be used in VR.
Follow tutorial to make a test body

folowing tutorial with a youtube video we set up an rig body with 3D model download from unity asset store.
first step will be import animation rigging from unity packages.
There are many easy functions in the package to set up the rigging for us. simply click on "Rig Setup" and "Bone Renderer Setup", the rigging will be set up.
Here you go, we can see the bones of the rigging, and all parts are connected to each other. Even if we pull some of his body parts, it will still be connected.
However, the arms are not connected as we expected. Following the tutorial, we added "Two Bones IK Constraint" to help us determine the position of the arm while we move the hand.
Before we go in our VR body, we need to set Constraints for the Vr controller to align with the body. For this part, we need to write our own script.
Now we can move around in our VR body without any problem.
