UI and Interaction
There are two UIs incorporated in the design. One is directly in front of the player, the other is displayed when the player turns towards an interactable object. The first displays the status of the player, the second displays the description of the object.

Shader graphs are also used to create the banding visual effect.

Multiple scripts are written for different types of interactable objects.
